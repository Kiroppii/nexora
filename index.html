Abstract

This project modernizes barangay certificate requests by replacing manual, paper-based processing with a structured web workflow. Residents can submit requests digitally, while barangay staff can validate, approve, reject, and notify residents more efficiently.

The goal is to reduce processing delays, minimize repetitive visits to the barangay office, and improve service transparency for all stakeholders.

System Workflow (BPMN)

Request flow: Resident submits a certificate request, system validates data, staff reviews submission, then approval or rejection is issued. Approved requests move to certificate generation and release notification.
